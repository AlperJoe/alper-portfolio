Pedro!

Online Multiplayer Card Game — Lua / LÖVE2D

Pedro is a fully developed online multiplayer card game implemented almost entirely in pure Lua, using LÖVE2D for rendering and a custom TCP server for real-time multiplayer gameplay.

The project focuses on deterministic game logic, explicit system design, and low-level control over networking, synchronization, and turn-based flow instead of relying on high-level engines or gameplay frameworks.

  • Custom TCP server for real-time multiplayer sessions and game rooms
  • Deterministic client-server architecture with authoritative game-state handling
  • Rule-based bots for offline play and multiple game modes
  • Custom turn / phase system with complex card and skill interactions
  • iOS deployment and performance optimization for production release
